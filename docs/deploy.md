## 수동 Docker 명령

```bash
# 빌드 + 시작
docker compose up -d --build

# 상태
docker compose ps

# 로그
docker compose logs -f

# 컨테이너 제거 (데이터 유지)
```

## Rollback

```bash
docker compose down
git checkout HEAD~1
docker compose up -d --build
```
